## Implemented directly (PR `improve/audit-fixes-wave2`, 2026-07-18)

Deep-audit findings fixed in the same session that produced plans 019-023:

- Redaction integrity: manually-whiteouted text ops now mask their covered
  originals in CSV/TXT/XLSX exports and Find & Replace (mirrors the PDF writer).
- Find & Replace: length-preserving case fold — no more index corruption on
  locale-expanding characters (Turkish "İ").
- Atomic writers: stamp/annotation/form-field probe text encodability before
  painting, so a skipped operation leaves no stray empty box.
- State: undo history survives page insert/delete/rotate; session fingerprint
  stays stable across page ops (no more orphaned IndexedDB rows); group
  duplicate keeps the whole new group selected; `pointercancel` discards
  instead of committing a half-finished drag.
- Resilience: top-level React `ErrorBoundary` with an autosave-aware fallback.
- Security: `sanitizeUrl` routes `mailto:` through the strict email validator
  and rejects embedded credentials; CSV/XLSX formula guard covers leading
  whitespace; CI gets `permissions: contents: read`, a pinned bun version, and
  a Playwright browser cache; `video:typecheck` added to CI.
- Perf: `tailwind-merge` removed (~100 kB dead source; `cn` is plain clsx);
  the text-layer MutationObserver filters mutation targets so drag frames no
  longer force span-scan reflows; `/assets/*` get immutable cache headers.
- Hygiene: Remotion showcase no longer depicts the removed Table tool;
  `plan.md` carries a SUPERSEDED banner; favicon + OG/Twitter meta added;
  ToolRibbon menus/history dialog close on Escape; `@vitest/coverage-v8` pin
  aligned; stale "e2e resize suite" v8-ignore comments replaced with real
  branch coverage; new `textRunGrouping` unit suite; corrupt-PDF rejection test.

## Findings considered and rejected

- **`PdfCanvas.tsx` decomposition (920 lines, ~10 concerns)**: real debt, but L effort with the highest regression surface in the app; postponed until a feature next forces a change there.
- **PageRail thumbnail virtualization**: valid at 100+ pages; deferred — typical documents are small, and `React.memo` already removed the repeated re-render cost.
- **Persisting undo history (`past`/`future`) to IndexedDB on autosave**: needs a product decision on whether undo must survive reload; not a pure code fix.
- **`react-select` replacement for the font picker**: bundle-size claim unmeasured; run `vite build` with a bundle analyzer before spending the effort.
- **78-file prettier reformat + `format:check` CI gate**: correct but belongs in its own dedicated PR to avoid drowning functional diffs.
- **`bun update jsdom` (undici advisories)**: dev-only exposure, zero production path; clear opportunistically.
- **React 18→19**: no cost to staying on 18 today; bundle with a future react-pdf/pdfjs-dist bump.
- **Multi-select alignment commands / CSV bulk form-fill / PNG export**: grounded direction ideas, not defects; PNG claim removed from README instead.
